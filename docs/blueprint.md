# **App Name**: Ventas Claras

## Core Features:

- Dashboard Overview: Friendly dashboard summarizing sales metrics, top clients, and best-selling products within specified date ranges.
- Client Management: A clear interface allowing salespeople to input leads with their full names, emails, telephone number, and birthdays. All information is saved.
- Sales Recording: Tool for keeping record of sold products with details, prices, sales date and expected restocking time.
- Customer Nurturing: Interface to create custom checklist-style tasks for each customer, such as following up on samples or referrals.
- Payment Tracking: Form that salespeople can use to save transaction dates, currency type, amounts, and statuses.

## Style Guidelines:

- Primary color: Light sky blue (#87CEEB), evoking a sense of calmness and trustworthiness, which will dominate the UI.
- Background color: Very light grayish-blue (#F0F8FF) to provide a soft, unobtrusive backdrop for content.
- Accent color: Soft lavender (#E6E6FA) to highlight interactive elements and calls to action without overwhelming the user.
- Body and headline font: 'PT Sans', a humanist sans-serif, will be used for a clear and readable text experience.
- Simple design to give beginning users an uncomplicated and guided user experience.
- Simple and easily understandable icons for key functions. Make them subtly animated to help beginning users know where to click.
- Use soft and subtle animations (not too flashy!) when payments get logged, sales recorded, etc. This subtly helps users keep track of new actions and provide a bit of fun.
